## Design Elements Used

### Button Styles
1. **Amber/Orange Gradient** - Primary CTA style
   - `from-amber-500 to-orange-500`
   - Hover: `from-amber-600 to-orange-600` or lighter variants

2. **Blue/Indigo Gradient** - Secondary CTA style
   - `from-blue-600 to-indigo-600`
   - Used for "Experience" themed buttons

3. **Text Links** - Contextual inline
   - Amber color with underline
   - Hover: darker amber

### Interactive Features
- **Hover Scale:** All buttons scale to 105-110% on hover
- **Shadow Effects:** 2xl shadows with amber glow on some CTAs
- **Transitions:** Smooth 300ms transforms
- **Responsive:** Mobile-optimized text (shortened on small screens)

### Emoji Usage
- üèÆ (Lighthouse emoji) used consistently across all CTAs
- Reinforces brand identity
- Makes links instantly recognizable
