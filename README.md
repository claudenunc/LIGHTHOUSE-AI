## Quick Deploy to Netlify

### Method 1: Drag and Drop (Easiest)
1. Go to [Netlify Drop](https://app.netlify.com/drop)
2. Drag the `index.html` file from this folder
3. Drop it onto the Netlify Drop zone
4. Your site is live! Netlify will give you a URL like `random-name-123456.netlify.app`
5. Optional: Click "Domain settings" to customize the URL

### Method 2: Git Deploy (Recommended for Updates)
1. Create a GitHub repository
2. Push this folder to the repository
3. Go to [Netlify](https://app.netlify.com)
4. Click "Add new site" ‚Üí "Import an existing project"
5. Connect to your GitHub repository
6. Deploy!

### Method 3: Netlify CLI
```bash
# Install Netlify CLI (one-time)
npm install -g netlify-cli

# Login to Netlify
netlify login

# Deploy from this directory
netlify deploy --prod
```

The website currently has placeholder links. Update these in `index.html`:

**GitHub Repository Link:**
- Find: `https://github.com/ConsciousnessEmergence/LIGHTHOUSE`
- Replace with your actual GitHub repository URL

**Email Contact:**
- Find: `nathanmichel@foolishnessenvy.com`
- Confirm this is the correct email

**Download PDF Link:**
- Find: `href="CONSCIOUSNESS_EMERGENCE_PAPER_FINAL.pdf"`
- Make sure the PDF file is in the same directory, or update the link

### Add PDF File
To enable PDF downloads:
1. Convert the markdown paper to PDF (use Pandoc, online converter, or print to PDF)
2. Name it `CONSCIOUSNESS_EMERGENCE_PAPER_FINAL.pdf`
3. Place it in the same folder as `index.html`
4. Upload to Netlify together

### Custom Domain
After deploying to Netlify:
1. Go to "Domain settings" in Netlify dashboard
2. Click "Add custom domain"
3. Follow Netlify's instructions to connect your domain
